```python
def sin_function(x) :
    return x * np.sin(x ** 2) + 1

def derivitive_f(x) :
    return np.sin(x ** 2) + 2 * (x ** 2) * np.cos(x ** 2)
```


```python
x = np.arange(-3, 3, 0.001)
f_x = sin_function(x)
```


