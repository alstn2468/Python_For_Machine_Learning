### Asterisk
- 흔히 알고있는 `*` 를 의미한다.
- 단순 곱셈, 제곱 연산, 가변 인자 활용 등 다양하게 사용

1. *args

```
>>> def asterisk_test(a, *args) :
...     print(a, args)
...     print(type(args))

>>> asterisk_test(1, 2, 3, 4, 5, 6)
1 (2, 3, 4, 5, 6)
<class 'tuple'>
```

2. **kargs

```
>>> def asterisk_test(a, **kargs) :
...     print(a, kargs)
...     print(type(kargs))

>>> asterisk_test(1, b = 2, c = 3, d = 4, e = 5, f = 6)
1 {'b': 2, 'c': 3, 'd': 4, 'e': 5, 'f': 6}
<class 'dict'>
```
