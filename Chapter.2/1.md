### 파이썬 스타일 코드
- 파이썬 스타일의 코딩 기법
- 파이썬 특유의 문법을 활용하여 효율적으로 코드 표현
- 고급 코드를 작성할 수록 더 많이 필요
- 예) 여러 단어들을 하나로 붙일 때

```
>>> colors = ['red', 'blue', 'green', 'yellow']
>>> result = ''
>>> for s in colors :
...    result += s

>>> result
'redbluegreenyellow'        

>>> colors = ['red', 'blue', 'green', 'yellow']
>>> result = ''.join(colors)
>>> result
'redbluegreenyellow'
```
